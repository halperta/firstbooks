Reading the First Books

This website is a temporary staging site for books transcribed for the Reading the First Books project. The project used the Ocular Optical Character Recognition system to transcribe fifty books from the Primeros Libros collection. Follow the links to see page transcriptions, or to download the transcription for an entire book. The transcriptions have not been reviewed or undergone post-processing of any kind.

Download the transcribed corpus. [.zip]
View sample transcriptions from the Reading the First Books symposium.

For more information about the project or the data, contact Hannah Alpert-Abrams (halperta@gmail.com).
